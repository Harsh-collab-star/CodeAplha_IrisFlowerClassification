# Iris Flower Classification

## Overview
This project uses machine learning to classify Iris flower species based on physical measurements.

## Dataset
Iris dataset containing:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Models Used
- Logistic Regression
- K-Nearest Neighbors
- Decision Tree

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- Classification Report

## Results
Logistic Regression achieved the highest accuracy (~97%).

## Tools
Python, Scikit-learn, Pandas, NumPy, Matplotlib

## How to Run
pip install -r requirements.txt  
python iris_classification.py
